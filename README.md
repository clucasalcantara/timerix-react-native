# react-native-timer-hooks

simple hooks for timer

## Installation

```sh
npm install react-native-timer-hooks
```

## Usage

```js
import TimerHooks from "react-native-timer-hooks";

// ...

const result = await TimerHooks.multiply(3, 7);
```
